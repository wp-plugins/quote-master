Adds a random quote to the top-right corner of the Dashboard, as well as includes a quote widget.

== Description ==
Adds a quote, chosed from a list of famous quotes, to the top-right corner of the Dashboard.  Also features an optional widget that will display random quotes on the website.

* WordPress: 2.8 or newer

= Basic Installation =
* Upload plugin dir to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Set plugin parameters for your needs
