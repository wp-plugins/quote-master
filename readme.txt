== Description ==
This adds a widget that displays random quotes to your visitors.  Also features easy to use shortcode: [quotemaster], so you can have random quotes anywhere.  This also has an option to add random quotes to the admin dashboard which can be turned off in the new options page..  Now features more than 100 quotes.

50% of donations are given to local school events and organizations.  The local marching band is in need of new marching uniforms, please help me sponsor them.

== Installation ==

= Minimal Requirements =
* WordPress: 2.8 or newer

= Recommended Requirements =
* WordPress: 2.8.2 or newer

= Basic Installation =
* Upload plugin dir to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Set plugin parameters for your needs
